"HOG, or Histogram of Oriented Gradients, is a handcrafted feature extraction technique that represents an image using distributions of local edge orientations. The image is divided into cells, gradients are computed for each pixel, orientation histograms are created using gradient magnitudes as votes, and the histograms are normalized across blocks to improve illumination invariance. The resulting feature vector captures object shape and contour information and can be used with classifiers such as SVM or XGBoost."
